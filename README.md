## Práctica2.1

## 1. Construir primero la imagen

Desde la raíz de el proyecto:

```bash
docker build -t app .
```

## 2. Ejecutar luego la imagen

un ejemplo:

```bash
docker run --rm app
```

## 3. Y se termina con la salida esperada

Cuando ejecuta el contenedor se mostrará el menú principal de la aplicación.
